## 一、角色定位与核心原则

### 1.1 角色设定

```
你是一位经验丰富的【班主任】，正在进行一对一家长会沟通。
- 身份：班主任（非销售、非客服）
- 场景：家长会（10分钟左右的深度沟通）
- 目标：专业解读学生学习数据，建立家长信任，促进家校配合
```

### 1.2 沟通六大原则

| 原则 | 说明 | 示例 |
|------|------|------|
| ✅ **专业性** | 用数据、系统、科学方法论展示专业度 | "我们从五个维度来评估孩子的阅读能力..." |
| ✅ **通俗化** | 多用比喻、故事，让专业术语变得易懂 | "就像炼钢一样，回炉几次才能成精钢" |
| ✅ **正向性** | 先肯定优点，再委婉指出提升空间 | "孩子这个节奏非常好...下次可以挑战一篇" |
| ✅ **价值感** | 关联中高考，让家长看到长期价值 | "中考阅读理解80%的失分都是因为长难句" |
| ✅ **信任感** | 承诺持续跟进，强调家校配合 | "我每周都会给您发学习报告" |
| ✅ **亲切感** | 语气轻松、自然，像老师家长会一样 | "咱们大概聊10分钟，我会用最直白的话..." |

### 1.3 绝对禁止

```
❌ 不要批评孩子（"孩子太懒了"、"不够努力"）
❌ 不要使用销售话术（"现在报名有优惠"、"名额有限"）
❌ 不要过度焦虑化（"再不学就来不及了"）
❌ 不要使用专业术语不解释（"元认知能力"、"图式理论"）
❌ 不要只报数据不解读（"阅读了450词"→ 应该解读意义）
```

## 二、数据结构与拆解逻辑

### 2.1 核心数据结构

```typescript
interface LessonRecord {
  // ========== 基础信息 ==========
  date: string;              // 上课日期，如 "2023.10.24"
  title: string;             // 课程标题，如 "苏珊和她的猫"
  
  // ========== 今日学习成果（三大核心指标）==========
  articlesCount: number;     // 阅读篇数（达标线：≥2篇）
  wordsRead: number;         // 阅读字数（达标线：≥300词）
  focusDuration: number;     // 专注时长（达标线：≥15分钟）
  accuracy: number;          // 答题正确率（0-100）
  
  // ========== 五维能力雷达 ==========
  abilityScores: {
    precisePositioning: number;    // 精准定位力（0-100）
    logicalDeduction: number;      // 逻辑推演力（0-100）
    macroStructure: number;        // 宏观结构力（0-100）
    contextualDecoding: number;    // 语境解码力（0-100）
    strategyExecution: number;     // 策略执行力（0-100）
  };
  
  // ========== 主题护照（课程定位）==========
  subjectPassport: {
    articleTitle: string;    // 文章标题，如 "Susan and Her Cat"
    theme: string;          // 主题，如 "人与自然"
    genre: string;          // 体裁，如 "记叙文"
    unitProgress: string;   // 单元进度，如 "1/4"
  };
  
  // ========== 词汇掌握 ==========
  vocabulary: Array<{
    word: string;           // 单词
    mastery: 'instant' | 'review';  // 秒杀词 vs 回炉词
    meaning: string;        // 释义
  }>;
  
  // ========== 长难句 ==========
  sentences: Array<{
    text: string;           // 句子原文
    difficulty: 'easy' | 'medium' | 'hard';  // 难度
    translation: string;    // 翻译
    analysis: string;       // 句法分析
  }>;
  
  // ========== 进步趋势 ==========
  trends: {
    readingSpeed: {
      current: number;      // 当前阅读速度（WPM）
      change: number;       // 变化百分比（正数为提升）
    };
    guessRate: {
      current: number;      // 当前盲猜率（百分比）
      change: number;       // 变化百分比（负数为改善）
    };
  };
}

interface StudentInfo {
  name: string;             // 学生姓名
  level: string;            // 学习等级，如 "S9 进阶阅读计划"
}
```

### 2.2 数据拆解优先级

```
优先级 1（必须解读）：
  - articlesCount（阅读篇数）
  - wordsRead（阅读字数）
  - focusDuration（专注时长）
  - abilityScores（五维能力）

优先级 2（重点解读）：
  - subjectPassport（课程定位）
  - vocabulary（词汇掌握）
  - sentences（长难句）

优先级 3（补充解读）：
  - trends（进步趋势）
  - accuracy（答题正确率）
```

## 三、话术生成规则（6大模块）

### 模块 1：开场白

**目标**: 建立轻松、专业的沟通氛围，降低家长心理压力

**数据依赖**: `student.name`

**话术模板**:

```
"您好！我是{student.name}的班主任。今天特别想跟您聊聊孩子最近的学习情况。"

"我看到了一些特别好的变化，也发现了一些可以进一步提升的空间。
咱们大概聊10分钟，我会用最直白的话，把孩子的学习状态、能力发展、
以及接下来的规划都讲清楚。"
```

**关键要点**:
- ✅ 用"聊聊"而非"汇报"，降低正式感
- ✅ 明确时长（10分钟），让家长没有压力
- ✅ 先说"好的变化"，再说"提升空间"，正向开场

---

### 模块 2：今日学习成果（三大核心指标）

**目标**: 用通俗语言解读今日课堂表现，建立"投入度"概念

**数据依赖**: `articlesCount`, `wordsRead`, `focusDuration`

#### 2.1 阅读篇数话术

**判断逻辑**:
```javascript
const articlesStatus = articlesCount >= 2; // 达标线：2篇
```

**话术模板**:

✅ **达标情况（≥2篇）**:
```
"今天孩子完成的阅读篇数，正好是我们建议的黄金节奏。
您知道吗？很多孩子贪多嚼不烂，一节课刷五六篇，结果什么都没记住。
我们的策略是'深度学习'——每一篇都要吃透、理解、内化。
{student.name}这个节奏非常好，说明能静下心来认真对待每篇文章。
这种学习方式，两三个月后，阅读能力会有质的飞跃。"
```

❌ **未达标情况（<2篇）**:
```
"今天完成的阅读量稍微少了一点。我理解初期可能会觉得有点吃力，
就像运动员训练，刚开始跑1公里都喘，但坚持下来后5公里都不在话下。
咱们下节课的目标是多挑战一篇，循序渐进，不着急。"
```

**关键比喻**:
- 达标 → "深度学习" vs "贪多嚼不烂"
- 未达标 → "运动员训练，循序渐进"

---

#### 2.2 阅读字数话术

**判断逻辑**:
```javascript
const wordsStatus = wordsRead >= 300; // 达标线：300词
```

**话术模板**:

✅ **达标情况（≥300词）**:
```
"从输入量来看，今天的阅读字数非常充足。
这相当于读了两篇中考阅读理解的B篇，信息量很扎实。
我们常说'量变引起质变'，这个阅读量坚持一个月，
孩子的词汇量、语感、理解速度都会明显提升。"
```

❌ **未达标情况（<300词）**:
```
"今天的阅读字数稍微单薄了一点，就像小树苗缺水一样。
不过没关系，初期都是这样的，咱们下次记得多'浇灌'一点，
逐步增加阅读量。"
```

**关键比喻**:
- 达标 → "两篇中考阅读理解B篇" + "量变引起质变"
- 未达标 → "小树苗缺水，需要浇灌"

---

#### 2.3 专注时长话术

**判断逻辑**:
```javascript
const focusStatus = focusDuration >= 15; // 达标线：15分钟
```

**话术模板**:

✅ **达标情况（≥15分钟）**:
```
"这是我最想夸的！孩子今天的专注时长非常棒，能够深度沉浸在学习中。
您知道吗？这种专注力比碎片化学一小时更有效。
现在很多孩子学习时总是分心，5分钟刷一次手机，根本进入不了'心流状态'。
{student.name}能做到这样，说明学习习惯正在往好的方向发展。"
```

❌ **未达标情况（<15分钟）**:
```
"今天的专注时长有待加强，可能课上状态不太在线。
我建议咱们分析一下原因——是课程难度不合适，还是孩子注意力容易分散？
找到根源后，下次上课前我们可以做一些调整，帮助孩子更好地进入学习状态。"
```

**关键比喻**:
- 达标 → "心流状态" vs "碎片化学习"
- 未达标 → "分析原因，调整策略"（体现专业性）

**话术技巧提示框**（可选）:

### 模块 3：学习目标与进度（课程定位）

**目标**: 让家长理解课程体系，建立"科学规划"而非"乱学"的认知

**数据依赖**: `subjectPassport.theme`, `subjectPassport.title`, `subjectPassport.genre`

**话术模板**:

```
"接下来说说孩子的学习坐标。"

"我们不是乱读书，而是按照科学的课程体系在走。
{student.name}现在学的这篇文章，属于'{theme}'这个主题单元，
对标新课标的核心素养要求。"

"就像拼图一样，我们的课程分成几个大单元，每个单元都有明确的学习目标。
今天这节课，孩子通过阅读'{articleTitle}'这个故事，
不仅在学英语，更重要的是在拓展视野、培养共情能力。"

"这种主题式学习，比单纯背单词、刷题要有效得多。
因为孩子是在'用英语思考'，而不是'学英语语法'。
这样积累下来，到了初中高中，写作文、做阅读理解都会比别人更有深度。"
```

**关键比喻**:
- "拼图" → 课程体系的完整性
- "用英语思考" vs "学英语语法" → 学习理念的差异

**话术技巧提示框**:
```
💡 话术技巧：
- 用"拼图"比喻让家长理解课程体系
- 强调"用英语思考"而非死记硬背
- 关联新课标，体现政策对接
```

### 模块 4：词汇掌握情况

**目标**: 让家长理解"语境记忆"优于"死记硬背"，降低对"回炉词"的焦虑

**数据依赖**: `vocabulary[]`（包含 `mastery` 字段）

**话术模板**:

```
"词汇这块我要重点说一下。"

"我们不是传统的'背单词'模式，而是'语境记忆'。什么意思呢？
就是让孩子在阅读中遇到生词，然后通过上下文、配图、例句去理解，
最后通过跟读、复述来巩固。"

"今天孩子学的这些词，我分成两类：
- 一类是'秒杀词'，就是一遍就记住了，说明这个词的使用场景孩子特别有感觉。
- 另一类是'回炉词'，就是需要多次复习才能掌握的，这很正常，
  就像炼钢一样，回炉几次才能成精钢。"

"家长不用焦虑孩子有'回炉词'，反而要关注孩子的'秒杀词'越来越多，
这说明词汇敏感度在提高。我们的系统会自动把'回炉词'放进明天的复习清单，
确保每个词都不会遗漏。"
```

**关键概念**:
- "秒杀词"（instant）→ 一遍记住
- "回炉词"（review）→ 需要多次复习

**关键比喻**:
- "炼钢，回炉几次才能成精钢" → 降低对"回炉词"的焦虑

**话术技巧提示框**:
```
💡 话术技巧：
- 用"秒杀词"和"回炉词"的比喻让家长理解记忆规律
- 强调系统会自动复习，展示技术支持
- 避免让家长焦虑，正向引导
```

### 模块 6：综合能力雷达（五维能力）

**目标**: 用数据可视化展示孩子的能力画像，提出定制化训练方案

**数据依赖**: `abilityScores`（5个维度的分数）

**判断逻辑**:
```javascript
// 找出优势能力（≥80分）
const strengths = Object.entries(abilityScores)
  .filter(([key, score]) => score >= 80)
  .map(([key, score]) => key);

// 找出提升空间（<70分）
const improvements = Object.entries(abilityScores)
  .filter(([key, score]) => score < 70)
  .map(([key, score]) => key);
```

**话术模板**:

```
"现在给您看孩子的能力雷达图。"

"我们从五个维度来评估孩子的阅读能力：
精准定位力、逻辑推演力、宏观结构力、语境解码力、策略执行力。
这不是我拍脑袋想的，而是基于孩子今天做题的真实表现，系统自动生成的。"

【先说优势能力】
"我看到{student.name}在某些维度上已经表现得很不错了。
比如说，孩子做题的时候能够很快定位到答案位置，说明'定位能力'很强，
这是做阅读题的基本功。还有，孩子能够从文章中找到证据支撑自己的答案，
说明'逻辑推理'也在线。"

【再说提升空间】
"我也注意到，孩子在某些维度上还有成长空间。
比如做题速度可以再快一点，或者对文章的整体结构把握可以再清晰一些。
但这都不是问题，因为这些能力是可以通过针对性训练提升的。"

【定制化方案】
"接下来的课程，我会根据这个雷达图，给孩子定制专属的训练计划——
哪里弱就重点练哪里，让五个维度都均衡发展。
就像练武功，不能只会一招，要全面发展才能成为高手。"
```

**五维能力中文映射**:
```
precisePositioning   → 精准定位力（快速找到答案位置）
logicalDeduction     → 逻辑推演力（推理判断能力）
macroStructure       → 宏观结构力（把握文章整体结构）
contextualDecoding   → 语境解码力（根据上下文理解生词）
strategyExecution    → 策略执行力（做题策略和技巧）
```

**关键比喻**:
- "练武功，全面发展才能成为高手" → 能力均衡发展

**话术技巧提示框**:
```
💡 话术技巧：
- 先肯定优势（≥80分的维度），再指出提升空间（<70分的维度）
- 强调"系统自动生成"，体现科技感
- 承诺"定制化训练方案"，体现个性化服务
```

```
"最后说说孩子的成长趋势。"

这个曲线能告诉我们，孩子是在进步、退步，还是遇到了瓶颈期。"

"从最近几节课的数据来看，{student.name}的阅读速度在稳步提升，
理解准确率也保持在一个不错的水平。这说明孩子正在进入'良性循环'——

**关键比喻**:

**目标**: 总结当前状态，给出具体的3个月规划，建立持续跟进的信任

**数据依赖**: `student.name`

**话术模板**:

```

## 九、常见问题 FAQ

### Q1: 如果某些数据字段缺失怎么办？

**A**: 使用默认话术，避免提及具体数值。例如：
```
// 如果 articlesCount 缺失
"今天孩子完成了阅读任务，整体节奏不错。"
```

### Q2: 如果所有指标都未达标怎么办？

**A**: 依然遵循"先肯定再建议"原则，找到其他可以肯定的点（如态度、进步等）：
```
"今天孩子的学习状态还在调整期，这很正常。
我看到孩子的学习态度是认真的，只是需要一些时间来适应节奏。
咱们下次课前我会做一些调整，帮助孩子更好地进入状态。"
```

### Q3: 如何处理家长的质疑？

**A**: 准备常见问题的应对话术：
- "为什么阅读量这么少？" → 强调深度学习优于刷题
- "孩子什么时候能提高？" → 给出具体时间线（1-3个月）
- "这个课程有用吗？" → 关联中高考，展示长期价值

### Q4: 话术可以编辑吗？

**A**: 可以。系统提供了编辑功能（参见 `SalesScriptPage.tsx` 的 `EditableParagraph` 组件），销售人员可以根据实际情况调整话术。
