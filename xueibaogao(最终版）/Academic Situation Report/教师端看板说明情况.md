# 教师端看板使用说明

## 功能概述

教师端看板允许教师通过输入学生ID查找学生的学情报告，查看最近一节课的详细报告，以及浏览历史课程记录。

## 主要功能

### 1. 学生搜索
- 在搜索框中输入学生ID（测试ID：1001）
- 点击"查找学生"按钮或按回车键
- 系统会显示该学生的信息和最近课程

### 2. 查看报告
- 找到学生后，可以点击"查看详情"查看最近一节课的完整报告
- 报告包含：
  - 学生基本信息
  - 课程摘要（阅读篇数、正确率、专注时长）
  - 五维能力雷达图
  - 能力分析和反馈
  - 盲猜率趋势
  - 课程详细信息

### 3. 历史课程记录
- 在报告页面底部可以看到历史课程导航栏
- 点击不同日期的课程可以切换查看不同课程的报告
- 在搜索页面也可以看到完整的历史课程列表

## 访问方式

### 方式一：通过模式选择器
1. 打开应用后，会显示模式选择界面
2. 点击"教师端"按钮进入教师看板

### 方式二：通过URL参数
在URL后添加 `?mode=teacher` 直接进入教师端
例如：`http://localhost:3000/?mode=teacher`

### 方式三：通过URL参数直接进入学生端
在URL后添加 `?mode=student` 直接进入学生端
例如：`http://localhost:3000/?mode=student`

## 界面说明

### 搜索页面
- 顶部显示"教师看板"标题
- 中间是学生搜索框
- 找到学生后显示最近课程卡片和历史课程列表

### 报告页面
- 顶部固定导航栏，可以返回搜索页面
- 显示学生基本信息和课程摘要
- 五维能力雷达图展示学生的综合阅读能力
- 能力分析和反馈提供教学建议
- 底部导航栏可以快速切换不同课程

## 技术实现

- 使用 React + TypeScript 构建
- 复用学生端的图表组件（雷达图、趋势图）
- 响应式设计，支持移动端和桌面端
- 使用 Tailwind CSS 进行样式设计
